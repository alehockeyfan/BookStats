📚 My Reading Stats

Upload your Goodreads export file to instantly generate your personalized literary dashboard.

🇬🇧 Quick Guide

  1. Language Selector: Use the dropdown in the top-right corner to switch between English and Italian at any time.
  2. Export from Goodreads: Log into goodreads.com, click on My Books, then click on Import/Export in the sidebar.
  3. Download: Click Export Library and download the CSV file from the link that appears.
  4. Load File: Drag or load the downloaded file using the orange button in the header above!
  5. Privacy: Your data is processed entirely inside your web browser. No files are uploaded to any external server.

🇮🇹 Guida Rapida

  1. Scelta Lingua: Usa il menu a tendina in alto a destra per cambiare la lingua tra Italiano e Inglese in qualsiasi momento.
  2. Esportare da Goodreads: Accedi a goodreads.com dal computer, clicca su My Books, quindi seleziona Import/Export nella barra laterale.
  3. Download: Clicca sul pulsante Export Library e scarica il file CSV generato tramite il link che appare.
  4. Carica File: Trascina o carica il file CSV appena scaricato usando il pulsante arancione qui sopra!
  5. Privacy: I tuoi dati sono elaborati interamente all'interno del browser. Nessun dato viene inviato a server esterni.
